This is a image search app! you can put on this url something and I will search images related with your input.

  1. You can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. You can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. You can get a list of the most recently submitted search strings.

The pattern to search is:

https://enigmatic-hamlet-53822.herokuapp.com/api/imagesearch/YOURSEARCHIMAGE?offset=NUMBEROFRESULTS

Ps.: The number of results have to be between 1 and 10

Ps1.: The number of results is optional

The pattern to get a list of the most recently submitted search strings:

https://enigmatic-hamlet-53822.herokuapp.com/api/latest/

Ps.: This will show you the last 10 search terms and the date.
